# Prompts — research

Reusable prompt snippets for **investigation & recommendations** on this project. Keep them versioned; pair AI-facing prompts
with a golden case in `llm-eval-harness`.

## Starters
- "Given [context in docs/], investigation & recommendations: … Constrain to [scope]. Enforce our standards (STACK.md)."
- "Before coding, restate assumptions and acceptance criteria for [backlog id]."

## Conventions
- Reference canonical docs by path; don't restate them.
- Prefer smallest-slice-first; surgical changes; ask when underspecified.
