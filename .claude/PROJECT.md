# PROJECT — AI Opportunity Intelligence Platform

> Discover AI trends before everyone else. Validate opportunities. Build faster.

## One-liner

An AI-powered intelligence platform that continuously monitors the AI ecosystem, deduplicates
signals into **trends**, attaches a consistent **10-dimension opportunity scorecard + action plan**,
and delivers them via dashboards, alerts, briefs, reports, workspaces, and an API.

## Not

A news site · an AI-tools directory · a Google-Trends clone. The value is the synthesis, scoring,
and "what to do next" layer.

## Core loop

`ingest (sources) → cluster into Trend → score (10 dims + composite) → action plan → deliver`

## Personas (priority)

Indie Builder → Founder → Creator; expansion: Agency, Investor, Org Admin. See
[docs/00-discovery/PERSONAS.md](../docs/00-discovery/PERSONAS.md).

## North-star

**Weekly Acted-On Opportunities** — users who take a tracked action (save/build, export, alert,
API key) on a scored opportunity per week.

## Status

Docs (phases 1–22) complete; monorepo live; critical path (ingest→score→persist→serve→render)
running in-browser; CI/CD + releases automated. See [ROADMAP.md](ROADMAP.md).
